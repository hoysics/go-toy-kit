# go-toy-kit

带有实验性质的Golang工具箱。

探索非CURD的各种有趣小项目，部分以装逼/KPI为目的。虽然命名有玩具箱的含义，所有小工具都以可以在生产环境实用为目标。

## 内容列表

本工具箱包含以下内容（不定期挖新坑或埋部分坑）：

1. - [ ] 并发队列
2. - [ ] 协程池
3. - [ ] 分布式锁
4. - [ ] 基于内存的消息队列
5. - [ ] grpc/gin/gorm + 基于 Redis 的熔断限流和
6. - [ ] grpc/gin/gorm + 基于 Prometheus 采集数据的熔断限流和降级
7. - [ ] 缓存模式
